# MERN Boilerplate

A simple MERN boilerplate. 

Starts you out with:
<br>
  -Mongoose.js with a very simple user schema and model
<br>
  -Express Server that serves your build bundle
<br>
  -A small react app
<br>
  -React Router
<br>
  -CSS and SCSS module loading
<br>
  -Redux example which comes commented out
<br>
